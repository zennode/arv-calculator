Auto-Fetch Comps (RentCast API)

Sign up free at rentcast.io (50 lookups/month). Paste your API key below. A proxy URL is needed because browser security blocks direct API calls — see setup instructions.
How to set up the free proxy (5 min)
  1. Go to dash.cloudflare.com and create a free account
  2. Click Workers & Pages → Create → Create Worker
  3. Name it arv-proxy, click Deploy, then Edit Code
  4. Replace all the code with:
    export default {
  async fetch(request, env) {
    if (request.method === 'OPTIONS') {
      return new Response(null, { headers: {
        'Access-Control-Allow-Origin': '*',
        'Access-Control-Allow-Methods': 'GET',
        'Access-Control-Allow-Headers': '*',
      }});
    }
    const url = new URL(request.url);
    const apiKey = url.searchParams.get('key');
    const endpoint = url.searchParams.get('endpoint');
    const params = url.searchParams.get('params');
    const r = await fetch(
      'https://api.rentcast.io/v1/' + endpoint + '?' + params,
      { headers: { 'X-Api-Key': apiKey } }
    );
    return new Response(await r.text(), {
      status: r.status,
      headers: {
        'Content-Type': 'application/json',
        'Access-Control-Allow-Origin': '*',
      }
    });
  }
}
  5. Click Save and Deploy
  6. Copy the worker URL (looks like https://arv-proxy.yourname.workers.dev) and paste it above
